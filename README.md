# 新建费曼练习原型

## 原型范围

本原型覆盖“制作费曼模板 / 新建费曼练习”和“费曼报告”两部分，页面样式参考现有 iTeach 小学工作台的浅色壳。

核心链路：

1. 在课后练习模块的集团练习列表中点击“新建练习”。
2. 在下拉菜单中选择“费曼练习”。
3. 进入“新建练习”命名弹窗。
4. 确认名称后进入全屏“新建费曼练习”创建弹窗。
5. 配置费曼内容和费曼规则，并提交质检。
6. 从列表进入“班级练习详情”，在费曼练习单元点击“学员报告及批改”。
7. 进入“陈一诺 · 费曼报告”，查看该学员的报告总览、引用任务、整体表现与费曼表现，并补充老师点评后发布。

## 已覆盖能力

- 集团练习列表页：左侧“集团练习”默认展开且可点击折叠；页面按“脑力与思维 / 0.FY26 / 1.夏 / B体系 / L4-B / 第2章-和差倍综合”目录展示，包含面包屑、移动/复制/删除、搜索、新建文件夹、新建练习下拉、三条作业包数据和分页。
- 班级练习详情页：首页左侧“绑定练习 / 班级练习详情”和列表首行“班级练习详情”均可进入；页面参考 iTeach 班级练习详情原图，展示课次、上课时间、已完成/进行中/未完成统计和学员完成情况表；费曼练习完成单元提供“学员报告及批改”入口。
- 命名弹窗：参考 iTeach 小命名弹窗样式。
- 全屏创建弹窗：参考单词听写/口语跟读创建流程。
- 费曼内容：学生侧提示语为题目、图片、视频共用配置；下方可切换题目、图片素材、视频素材三种学生侧展示内容。
- 题目来源：当题目来源切换为“从题库中引用”时，创建弹窗右侧展示选题面板，可筛选题库、查看题目列表并加入题目。
- 图片上传：图片内容类型默认无图片，需先手工选择本地图片，选中后再点击“上传图片”完成上传模拟。
- 当前创建示例：小学四年级数学“动物城单位换算秘籍”，支持学生围绕单位家族和换算方法进行费曼讲解；图片素材需由老师手工上传。
- 费曼规则：以对话方式确认规则，支持老师语音录入、上传视频录入、手工填写文字三种输入方式。
- 题库视频回显：当题目来源为“从题库中引用”时，规则区直接回显题库中配置好的讲解视频，等同于老师手工上传视频录入。
- 小学堂预览：模拟学生侧“我的练习”页面，顶部显示“我的练习”，中间依次展示学生侧提示语和当前题目/图片/视频内容，底部固定“提交练习”按钮。
- 费曼报告：从班级练习详情的费曼练习入口进入，围绕当前学员展示报告总览、引用任务、整体表现、费曼表现和老师点评；支持老师调整分数、编辑报告结论、查看原始录音、录入语音点评并发布报告，右侧提供“小学堂预览”。

## 交互说明

- 点击“新建练习”展开下拉菜单。
- 点击左侧“集团练习”可折叠/展开其下目录，默认展开。
- 点击左侧“绑定练习 / 班级练习详情”进入班级练习详情页。
- 点击列表首行“班级练习详情”进入“班级练习详情”页。
- 在“班级练习详情”页点击费曼练习单元中的“学员报告及批改”，进入单个学员报告页。
- 点击“费曼练习”打开命名弹窗。
- 点击命名弹窗“确定”进入全屏创建弹窗。
- 先填写“学生侧提示语”，再点击“题目 / 图片 / 视频”切换费曼内容类型。
- 右侧“小学堂预览”会实时同步学生侧提示语和当前内容素材，展示学生提交前看到的练习页面。
- 在“题目”内容下，将“题目来源”改为“从题库中引用”时，右侧从制作进度/小学堂预览切换为选题面板；点击选题面板任意区域后关闭面板并提示引用成功，题目来源保持为“从题库中引用”。
- 在“图片”内容下，默认显示未上传空态；点击“选择图片”选择本地图片，选中后再点击“上传图片”完成上传模拟。
- 点击“确认需求，生成规则”模拟在对话区生成规则草稿。
- 点击“确认规则”后，制作进度变为可提交质检。
- 题目来源为“从题库中引用”时，费曼规则区自动显示题库配置视频。
- 在“费曼报告”页按“报告总览 / 本次引用任务 / 陈一诺的整体表现 / 老师点评”纵向查看。
- 点击“播放原始录音”会弹出对话内容与播放控件，点击“录入语音”会弹出录音面板，完成后展示一行录音文件并支持播放、删除。
- 右侧“小学堂预览”按“整体表现 / 费曼表现 / 费曼点评 / 引用任务”的阅读顺序展示学生侧报告。
- 点击“发布报告”会模拟报告发布状态。

## 原型假设

- 本轮只做静态 HTML 原型，不接真实题库、AI、音视频上传和权限系统。
- 当前入口按“集团练习 / 脑力与思维”目录示例展开，后续可复用到校本练习和个人练习。
- 当前示例素材来自用户提供的四年级数学单位换算图片，作为静态原型内置资源展示。
- 模板提交质检后的质检流程和版本管理后续单独设计。
- 图标资产未接入真实 iTeach 资源，本轮用文字符号和 CSS 近似。
