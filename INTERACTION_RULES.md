# 新建费曼练习交互矩阵

## 路由表

| 页面 | 路由 | 入口 | 职责 |
|---|---|---|---|
| 集团练习列表 | `index.html` | 左侧课后练习 / 集团练习 / 脑力与思维 | 展示当前目录面包屑、练习列表和新建练习入口 |
| 班级练习详情 | `pages/class-detail.html` | 集团练习列表首行“班级练习详情” | 展示课次、完成统计、学员完成情况，并提供费曼练习报告入口 |
| 新建练习命名弹窗 | `index.html` 内模态 | 新建练习 / 费曼练习 | 输入费曼练习名称 |
| 新建费曼练习全屏弹窗 | `index.html` 内模态 | 命名弹窗确定 | 配置费曼内容和费曼规则 |
| 费曼报告 | `pages/report.html` | 班级练习详情 / 费曼练习 / 学员报告及批改 | 查看当前学员报告总览、引用任务、整体表现与费曼表现，补充老师点评并发布报告 |

## 入口矩阵

| 入口 | 打开内容 | 初始状态 | 提交结果 | 关闭行为 |
|---|---|---|---|---|
| 新建练习 | 练习类型下拉 | 下拉收起 | 可选择费曼练习 | 再次点击或选择后收起 |
| 左侧集团练习 | 课后练习菜单 | 默认展开 | 点击折叠或展开“集团练习”下的目录项 | 不跳转 |
| 左侧班级练习详情 | 绑定练习菜单 | 绑定练习默认展开 | 点击进入班级练习详情页 | `pages/class-detail.html` |
| 班级练习详情入口 | 集团练习列表首行 | 课后练习作业包 | 点击“班级练习详情”进入班级练习详情页 | `pages/class-detail.html` |
| 学员报告及批改 | 班级练习详情 / 费曼练习单元 | 首位已完成学员 | 点击进入该学员费曼报告页 | `pages/report.html` |
| 费曼练习 | 新建练习命名弹窗 | 默认“动物城单位换算秘籍” | 点击确定进入创建弹窗 | 取消/关闭返回列表 |
| 命名弹窗确定 | 全屏创建弹窗 | 草稿状态 | 配置并提交质检 | 关闭返回列表 |
| 学生侧提示语 | 费曼内容公共配置 | 默认示例提示语 | 作为题目、图片、视频三种内容类型的学生端统一引导语 | 不跳转 |
| 费曼内容类型切换 | 题目/图片/视频内容区 | 图片 | 切换学生侧预览素材类型，保留上方学生侧提示语 | 不跳转 |
| 小学堂预览 | 创建弹窗右侧 | 我的练习 | 顶部显示“我的练习”；中间依次展示学生侧提示语和当前题目/图片/视频内容；底部显示“提交练习”按钮 | 不跳转 |
| 题目来源切换 | 题目内容区 | 手动录入 | 选择“从题库中引用”时，右侧展示选题面板；点击选题面板任意区域后关闭面板、提示“从题库中引用试题成功”，并保持题目来源为“从题库中引用” | 不跳转 |
| 图片上传 | 图片内容区 | 默认无图片 | 点击“选择图片”打开本地文件选择；选中后显示预览和文件名并启用上传；上传后状态变为已上传 | 不跳转 |
| 对话生成规则 | AI 规则工作台 | 待生成规则 | 支持语音录入、上传视频录入、手工填写文字，生成并确认规则后可提交质检 | 不跳转 |
| 题库视频回显 | 费曼规则区 | 题目来源为手动录入时隐藏 | 题目来源为“从题库中引用”时，自动回显题库配置视频作为规则输入素材 | 不跳转 |
| 费曼报告 | 班级练习详情 / 费曼练习 | 默认进入陈一诺报告 | 查看单个学员报告、批改并发布小学堂报告 | 返回班级练习详情 |
| 单个学员报告：报告总览 | 报告顶部 | 展示陈一诺综合评分、知识准确性、表达完整性和结论 | 手工加减分、编辑结论 | 不跳转 |
| 单个学员报告：本次引用任务 | 报告第一段 | 展示该学员本次引用的费曼任务 | 查看试题、图片/视频素材和判别依据 | 不跳转 |
| 单个学员报告：整体表现 | 报告第二段 | 默认展示陈一诺报告 | 查看该学员知识点掌握、费曼表现维度图和作答证据，可播放原始录音 | 不跳转 |
| 单个学员报告：老师点评 | 报告第三段 | 默认已有文字点评草稿 | 输入文字点评、录入语音、完成后播放或删除录音文件 | 不跳转 |
| 小学堂预览 | 右侧手机预览 | 默认展示陈一诺报告 | 按整体表现、费曼表现、费曼点评、引用任务顺序预览学生侧内容 | 不跳转 |

## 状态矩阵

| 状态 | 触发 | 页面文案 | 可用动作 |
|---|---|---|---|
| 列表浏览 | 打开页面 | 集团练习 / 脑力与思维 / 0.FY26 / 1.夏 / B体系 / L4-B / 第2章-和差倍综合（共3个） | 返回、移动、复制、删除、搜索、新建文件夹、新建练习 |
| 命名中 | 点击费曼练习 | 新建练习 | 输入名称、取消、确定 |
| 草稿 | 进入创建弹窗 | 草稿 | 保存草稿、学生侧预览 |
| 规则待确认 | 未确认费曼规则 | 待确认 | 语音录入、上传视频录入、手工填写文字、确认需求生成规则 |
| 规则已确认 | 点击确认规则或确认引用 | 已确认 | 提交质检 |
| 已提交质检 | 点击提交质检 | 模板已提交质检，待发布 | 后续由质检页承接 |
| 报告待审核 | AI 初评完成 | 待老师确认 | 查看当前学员引用任务、查看 AI 点评、调整分数、补充老师点评、发布报告 |
| 报告已发布 | 点击发布报告 | 已发布给家长 | 家长端可见 |

## 模态矩阵

| 模态 | 类型 | 打开入口 | 主要动作 | 关闭行为 |
|---|---|---|---|---|
| 新建练习命名弹窗 | FormModal | 新建练习 / 费曼练习 | 确定 | 返回练习列表 |
| 新建费曼练习全屏弹窗 | FullScreenCreationModal | 命名弹窗确定 | 保存草稿、学生侧预览、提交质检 | 返回练习列表 |
| 原始作答录音 | Dialog | 作答证据 / 播放原始录音 | 查看对话内容与播放控件 | 关闭返回报告 |
| 录入语音点评 | Dialog | 老师点评 / 录入语音 | 完成录制 | 生成录音文件行并返回报告 |

## 关键操作

| 操作 | 位置 | 业务含义 |
|---|---|---|
| 新建练习 / 费曼练习 | 练习池右上角 | 将费曼练习作为现有练习类型的一种进入 |
| 确认需求，生成规则 | 对话生成规则 | 从教师文字、语音或视频描述中提取练习目标、追问策略、评分维度和报告依据 |
| 确认规则 | 对话生成规则 | 教师确认 AI 草稿可作为费曼练习规则 |
| 提交质检 | 全屏创建弹窗底部 | 将模板提交给质检/发布流程 |
| 发布报告 | 单个学员报告顶部 | 老师确认该学员 AI 点评和手工点评后，将学生端报告开放给家长 |
| 录入语音 | 老师点评 | 为学生/家长端补充老师语音点评 |
